Check for unconnected network partsÂ¶

To check for network connectivity, weâll find shortest paths from one node to all other nodes.

[68]:

a_node = list(streets.nodes)[0] # Grab a node ID randomly

# Nodes that can be reached from a_node
connected_nodes_out = set(nx.shortest_path(streets, source=a_node, weight='length').keys())

# Nodes that can reach a_node
connected_nodes_in = set(nx.shortest_path(streets, target=a_node, weight='length').keys())

# Nodes that are full connected
connected_nodes = connected_nodes_out & connected_nodes_in

# The rest
unconnected_nodes = [node for node in streets.nodes() if node not in connected_nodes]
